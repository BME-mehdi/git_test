Hello World Programs in Different Languages

  • Python: 
    print("hello world")
  • C: 
    #include <stdio.h> 

int main() {
    printf("Hello, World!\n");
    return 0;
}
    • C++: 
    #include <iostream>        
int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
  • Java: 
    public class HelloWorld {
    public static void main(String[] args) {    
        System.out.println("Hello, World!");
    }
}
